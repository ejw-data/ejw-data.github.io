Content Policy

Purpose

This website is used as a basic profile page. A database driven website is being developed so that posting, searching, and other content features can be provided. The content of these pages is just a temporary location until the better website is complete.

Issues with this Site

I should also note that I looked at several different pre-made static website templates that were described as W3C compliant, but I believe that many common W3C features are not included in the website as well as other common features usually provided in the HTML comments. Some of the features not included are:

  • Accessibility
    • Coding page elements in order of importance (also helpful for SEO)
    • Labeling sub-menu navigation as navigation or proving options for screen readers to skip
    • Use of ARIA labels
    • Use of modern elements like section, figure, etc.
  • HTML comments
  • Few styled CSS elements
  • Place holders Google Analytics
  • Limited responsive design elements (2 screen size media queries)

Some of these features will be added over the summer of 2018 as part of an section of this site about best practices. For example, Google Analytics has already been added to each page.

Privacy

Information collected from this website is only used for improvement of content on different platforms and monitoring the utility of the website. When the database driven website is is introduced, the privacy policy will be updated to include responsibility statements regarding the use of registration information.

My Posts

As compared to other websites, a difference that may be noticed with some of my future writings is that I often post information about whether the paper I have written is an opinion paper or position paper. The distinctions are:

  • Opinion paper: reason-based but not necessarily empirical-based; uses emotional appeal
  • Position paper: empirically based but with analysis of evidence that may be skewed for a particular argument
